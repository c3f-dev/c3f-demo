What is This?

Welcome to our demo! This is a page for demonstrating practical covert communication between isolated browser instances via GPU command queue contention.

In this work, we discoverd a novel covert channel due to GPU command queue contention and then design a framework, called C3F, for both cross-mode and cross-browser covert communication.

Instructions

This page will be the receiver in the demo. Now, please keep this page here (i.e., not close it until finished).

Then, please open this link https://c3f-dev.github.io/c3f-demo/sender.html in any mode (normal or incognito) in a browser you like, e.g., Chrome, Edge, Opera, Firefox, etc..

That page will be the sender. By following the instructions there, the sender will covertly communicate with the receiver and you will see the message in the box below.
